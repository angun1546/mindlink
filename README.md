# 마음-이음 (Mind-Link) 개발 기록

AI와 게이미피케이션을 활용한 초등 교사 전용 상담 시간 확보 및 업무 자동화 플랫폼 개발 프로젝트입니다.

## 🚀 개발 철학 (Senior Developer's Note)
"기술은 결코 인간을 대체할 수 없으며, 단지 인간이 '진짜 소중한 것'에 집중할 수 있도록 도와야 합니다." 
이 프로젝트는 초등 교사분들이 행정 업무의 늪에서 벗어나 아이들의 눈을 한 번 더 맞출 수 있는 시간을 만들어드리는 것에 초점을 맞췄습니다.

## 🛠 기술 스택
- **Framework**: React 19 (Vite) - 최신 리액트 기능을 통한 성능 최적화
- **Animation**: GSAP - 시니어급 퀄리티의 부드럽고 인터렉티브한 UI 구현
- **Styling**: Vanilla CSS (Responsive) - 외부 라이브러리 의존도를 낮춘 견고한 스타일링

## 📋 주요 개발 완료 사항

### 1. 프로젝트 기반 구축 ✅
- Vite 기반의 고성능 리액트 환경 설정
- GSAP 애니메이션 라이브러리 연동 및 전역 레이아웃 설계

### 2. UI/UX 디자인 (Chapter 3 반영) ✅
- **메인 홈**: 학급 전체 정서를 한눈에 보는 '감정 기상청' 대시보드
- **상담 키트**: 학생 발화를 실시간으로 분석하여 감정 지표를 보여주는 UI
- **결과 리포트**: 상담 내용을 AI가 요약하고 인사이트를 도출하는 결과 화면

### 3. 지능형 핵심 기능 (Chapter 4 반영) ✅
- **감정 기상청 (Mood Check-in)**: 매일 아침 아이들의 기분을 날씨로 체크
- **마음 지기 NPC (Care Bot)**: 24시간 아이들의 이야기를 들어주는 AI 챗봇
- **세이프티 가디언 (Red-Flag Alert)**: 위기 징후 감지 시 즉각적인 교사 알림
- **트러스트 로그 (Trust Log)**: 블록체인 기반 보안으로 상담 데이터의 신뢰성 확보

### 4. 시니어급 디테일 ✨
- **GSAP 애니메이션**: 화면 전환 시 시각적 피로도를 줄이는 부드러운 트랜지션
- **완벽한 반응형**: PC, 태블릿, 모바일 등 모든 기기에서 최적화된 화면 제공
- **안정적인 코드 구조**: 런타임 오류 방지를 위한 철저한 검토 및 최적화
